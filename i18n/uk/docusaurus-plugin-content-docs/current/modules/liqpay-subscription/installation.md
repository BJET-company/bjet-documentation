# Встановлення та активація

## Встановлення модуля

1. Перейдіть у меню **Додатки** (Apps).

2. Знайдіть модуль **Платежі LiqPay для підписок в Odoo**.

3. Натисніть **Встановити** (Install).

:::info Note
**Зверніть увагу!** Модуль **Платежі LiqPay для підписок в Odoo** розширює можливості модуля **Платежі LiqPay на Odoo вебсайті**, який має бути встановлений і налаштований, щоб активувати автоматизовані платежі за підпискою через LiqPay
:::

## Наступні кроки

Після встановлення модуля перейдіть до [Налаштування модуля](./configuration.md) для ознайомлення з функціоналом.
