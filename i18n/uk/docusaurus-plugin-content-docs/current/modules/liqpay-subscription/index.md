# Платежі LiqPay для підписок в Odoo

**Технічна назва модуля:** `bjet_liqpay_subscription`

## Для чого цей модуль?

Модуль **Платежі LiqPay для підписок в Odoo** дозволяє налаштувати **періодичні списання коштів** із картки клієнта через платіжну систему **LiqPay**:
- при оформленні підписки на вебсайті Odoo;
- при оформленні підписки через **Посилання платежу (Payment Link)** з Рахунку (Invoice) або Замовлення на продаж (Sales Order).

Після успішної **першої оплати (передоплати)** система:
- реєструє підписку в LiqPay з такими параметрами:
  - дата початку дії (дата SO/Invoice);
  - сума списання;
  - періодичність стягнення (місяць, рік);
  - дата закінчення підписки;
- в Odoo автоматично створює та зв'язує між собою:
  - Рахунок (Invoice) (рахунок за перший період);
  - Платіжна операція (Payment Transaction);
  - Платіж (Payment).

При кожному наступному списанні за підпискою в Odoo знову створюються:

- новий Рахунок;
- новий Платіж;
- нова Платіжна транзакція.

![Діаграма процесу оплати підписки](/img/liqpay-subscription/ua-subscription-payment-flow-diagram.png)![Деталі плану повторюваних платежів у Замовленні на продаж](/img/liqpay-subscription/ua-sales-order-recurring-plan-details.png)

У платіжній транзакції та платежі зберігається **унікальний ID платежу LiqPay**, який може використовуватися для узгодження банківських виписок.

:::info Примітка
Модуль **Платежі LiqPay для підписок в Odoo** розширює можливості модуля **Платежі LiqPay на Odoo вебсайті**, який має бути встановлений і налаштований, щоб активувати автоматизовані платежі за підпискою через LiqPay.
:::
