# Аналоги компонентів виробництва з пріоритетами

## Для чого цей модуль?

У виробництві зупинка через нестачу матеріалу — це:
- зрив термінів
- простої обладнання
- додаткові витрати
- невдоволені клієнти

Цей модуль дозволяє автоматично замінювати дефіцитні компоненти на заздалегідь налаштовані аналоги з урахуванням пріоритету та коефіцієнта заміни.

## Що ви отримуєте

- Виробництво не зупиняється через дефіцит
- Система сама підбирає замінники за пріоритетами
- Підтримується часткова заміна з кількох замінників
- Повна прозорість — усі зміни фіксуються у замовленні на виробництво
- Контроль пріоритетності замінників
