# Встановлення та права доступу

## Встановлення модуля

1. Перейдіть у меню **Додатки** (Apps).

2. Знайдіть модуль **Аналоги компонентів виробництва з пріоритетами**.

3. Натисніть **Встановити** (Install).

:::info Примітка
Якщо модуль Manufacturing [mrp] ще не встановлений — він буде встановлений автоматично.
:::

## Доступи

Щоб працювати із замінниками, користувач повинен мати:
- право **"Дозволити аналоги (замінники)(MRP)"**
- стандартний доступ до **створення/редагування товарів**

Без цього вкладка замінників у товарі просто не відображатиметься.

### Як надати право (для адміністратора)

1. Перейти: **Налаштування → Користувачі та компанії → Користувачі**
2. Відкрити потрібного користувача
3. У вкладці **Права доступу** активувати чекбокс: **Дозволити аналоги (замінники)(MRP)**
4. Зберегти зміни

![Права доступу з дозволом Дозволити аналоги (замінники)(MRP)](/img/mrp-substitution/user-access-rights-ua.png)
