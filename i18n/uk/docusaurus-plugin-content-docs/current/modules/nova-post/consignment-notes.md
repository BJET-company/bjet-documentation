# Розділ 5: Управління накладними

Нова модель **Накладні** дозволяє централізовано керувати всіма відправленнями.

![Список накладних Нової Пошти](/img/nova-post/consignment-notes-ua.png)

1. Перейдіть у меню **Нова Пошта** (доступно у Продажах, Закупівлях та Складі) ➔ **Накладні**.
2. Тут ви знайдете повну інформацію про кожну накладну:
   - **Напрямок:** Вхідна (для закупівель) або Вихідна (для продажів).
   - **Дані про накладний платіж:** Сума, платник та статус виплати коштів.
   - **Історія статусів:** Дата останнього оновлення стану посилки.
3. **Оновлення даних вручну:** Натисніть **Оновити статус** у картці накладної, щоб миттєво отримати найсвіжіші дані з API поза графіком синхронізації.
4. **Відстеження на сайті Нової пошти:** натисніть **Відстежити на Новій пошті**, щоб переглянути інформацію про ТТН безпосередньо на сайті Нової пошти.
