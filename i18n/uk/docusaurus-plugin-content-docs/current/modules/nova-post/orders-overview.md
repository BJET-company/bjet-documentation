# Розділ 2: Робота з Замовленнями та Складом

## 2.1. Відображення ТТН у списках замовлень Продажі та Закупівлі

У модулях **Продажі** та **Закупівлі** є колонки для швидкого моніторингу номеру та статусу ТТН:

- **Номер відстеження:** Відображає номери всіх ТТН, пов'язаних із замовленням.
- **Перевізник:** Метод доставки.
- **Статуси доставки ТТН:** Кольорові бейджи з поточним станом доставки (наприклад, «Прибув у відділення» або «Отримано»).

Ви можете шукати замовлення за номером ТТН або групувати їх за статусом доставки через стандартну панель пошуку.
