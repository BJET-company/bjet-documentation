# Встановлення та активація

## Встановлення модуля

1. Перейдіть у меню **Додатки** (Apps).

2. Знайдіть модуль **СКВ (Структура BOM / Ієрархія компонентів у виробництві)**.

3. Натисніть **Встановити** (Install).

:::info Примітка
Якщо модуль Manufacturing [mrp] ще не встановлений — він буде встановлений автоматично.
:::

## Перевірка встановлення

Після встановлення модуля з'являються нові вкладки:

1. Перейдіть у **Виробництво → Специфікації** та відкрийте будь-яку специфікацію — має з'явитися вкладка **СКВ**.
2. Перейдіть у **Виробництво → Замовлення на виробництво** та відкрийте будь-яке замовлення — має з'явитися вкладка **СКВ**.

## Наступні кроки

Після встановлення модуля перейдіть до [Використання модуля](./usage.md) для ознайомлення з функціоналом.
