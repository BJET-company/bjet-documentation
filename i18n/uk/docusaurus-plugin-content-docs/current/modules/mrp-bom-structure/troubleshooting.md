# Усунення несправностей

## Типові проблеми та рішення

### Проблема: Вкладка СКВ порожня

**Симптоми:** Вкладка СКВ відображається, але не містить жодних даних.

**Можливі причини:**
- Продукт не має призначеної специфікації (Bill of Materials)
- Специфікація не містить компонентів

**Рішення:**
1. Перейдіть у **Виробництво → Специфікації**
2. Переконайтеся, що для продукту існує специфікація з компонентами
3. Перевірте, що специфікація має статус, який дозволяє її використання

### Проблема: Категорія товару не відображається

**Симптоми:** Колонка "Категорія товару" порожня для деяких компонентів.

**Можливі причини:**
- Поле "Категорія товару" не заповнене на картці товару

**Рішення:**
1. Відкрийте картку товару компонента
2. Перейдіть на вкладку **Загальна інформація**
3. Переконайтеся, що поле **Категорія товару** заповнене
4. Збережіть зміни

### Проблема: Структура не оновлюється після зміни кількості

**Симптоми:** Після зміни кількості у замовленні на виробництво вкладка СКВ показує старі дані.

**Можливі причини:**
- Зміни не були збережені
- Кеш браузера

**Рішення:**
1. Збережіть замовлення на виробництво після зміни кількості
2. Оновіть сторінку (F5 або Ctrl+R)
3. Якщо проблема зберігається — очистіть кеш браузера

### Проблема: Глибоко вкладені компоненти не відображаються

**Симптоми:** Деякі рівні ієрархії компонентів не відображаються у дереві.

**Можливі причини:**
- Перевищено максимальну глибину вкладеності (20 рівнів)
- Циклічні посилання у специфікаціях

**Рішення:**
1. Перевірте глибину вкладеності специфікацій — модуль підтримує до 20 рівнів
2. Переконайтеся, що специфікації не містять циклічних посилань (компонент A посилається на B, який посилається на A)

### Проблема: Повільне завантаження при великих специфікаціях

**Симптоми:** Вкладка СКВ довго завантажується для продуктів зі складними специфікаціями.

**Можливі причини:**
- Велика кількість рівнів та компонентів у специфікації

**Рішення:**
1. Це очікувана поведінка для дуже складних специфікацій
2. Використовуйте згортання рівнів для покращення продуктивності
3. Зверніться до підтримки, якщо час завантаження перевищує 10 секунд
