# СКВ (Структура BOM / Ієрархія компонентів у виробництві)

**Технічна назва модуля:** `bjet_mrp_bom_structure`

## Для чого цей модуль?

У складному виробництві один виріб може містити:
- десятки компонентів
- підкомпоненти
- багаторівневі специфікації

У стандартному Odoo побачити повну структуру матеріалів одразу складно.

Модуль **СКВ (Структура BOM / Ієрархія компонентів у виробництві)** вирішує цю проблему та дозволяє миттєво переглянути повну ієрархію всіх матеріалів виробу.

## Що ви отримуєте

- Повну структуру матеріалів виробу в одному місці
- Відображення усіх рівнів багаторівневих специфікацій (1, 2, 3…)
- Швидку перевірку наявності компонентів
- Точне планування закупівель
- Автоматичний перерахунок кількості при зміні обсягу виробництва

Модуль працює у:
- **Замовленні на виробництво** (Manufacturing Orders)
- **Специфікації** (Bills of Materials)
