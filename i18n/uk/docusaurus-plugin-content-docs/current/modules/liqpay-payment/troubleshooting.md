# Усунення несправностей

## Типові проблеми та рішення

### Проблема: LiqPay не відображається як спосіб оплати на сайті

**Симптоми:** При оформленні замовлення на вебсайті LiqPay відсутній серед доступних способів оплати.

**Можливі причини:**
- Провайдер LiqPay у стані «Відключено»
- Не заповнені облікові дані (публічний або приватний ключ)
- Валюта замовлення не підтримується (підтримуються лише UAH, USD, EUR)

**Рішення:**
1. Перейдіть у **Бухоблік → Конфігурація → Платіжні провайдери**
2. Відкрийте провайдер **LiqPay**
3. Переконайтеся, що стан провайдера — **«Увімкнено»** або **«Тестовий режим»**
4. Перевірте, що заповнені поля **Публічний ключ** та **Приватний ключ** на вкладці **Облікові дані**
5. Переконайтеся, що валюта замовлення — UAH, USD або EUR

### Проблема: Помилка автентифікації при оплаті

**Симптоми:** Після переходу на сторінку оплати LiqPay відображається помилка або порожня сторінка.

**Можливі причини:**
- Невірний публічний або приватний ключ
- Ключі від іншого середовища (тестові ключі у продакшн-режимі або навпаки)

**Рішення:**
1. Увійдіть до кабінету мерчанта LiqPay: [https://www.liqpay.ua/](https://www.liqpay.ua/)
2. Перейдіть у розділ **API** та скопіюйте актуальні **Public key** та **Private key**
3. Вставте ключі у відповідні поля провайдера в Odoo
4. Збережіть налаштування та повторіть оплату

:::warning Увага
Для тестового режиму використовуйте тестові ключі LiqPay (sandbox). Для продакшн-режиму — реальні ключі. Змішування ключів призведе до помилок підпису.
:::

### Проблема: Платіжне посилання не працює

**Симптоми:** При натисканні на посилання для оплати з рахунку або замовлення на продаж з'являється помилка або оплата не відкривається.

**Можливі причини:**
- Не увімкнено прапорець **Виставити рахунок на онлайн-оплату** в налаштуваннях бухобліку
- Провайдер LiqPay не активований
- Валюта документа не підтримується

**Рішення:**
1. Перейдіть у **Загальні налаштування → Виставлення рахунків (Бухоблік)**
2. Увімкніть прапорець **Виставити рахунок на онлайн-оплату**
3. Переконайтеся, що провайдер LiqPay у стані **«Увімкнено»** або **«Тестовий режим»**
4. Перевірте, що валюта рахунку або замовлення — UAH, USD або EUR
