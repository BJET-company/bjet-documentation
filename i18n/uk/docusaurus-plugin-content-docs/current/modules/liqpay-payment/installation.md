# Встановлення та активація

## Встановлення модуля

1. Перейдіть у меню **Додатки** (Apps).

2. Знайдіть модуль **Платежі LiqPay на Odoo вебсайті**.

3. Натисніть **Встановити** (Install).

:::info Примітка
Якщо модулі Виставлення рахунків [account] або Система оплати [payment] ще не встановлені — вони будуть встановлені автоматично.
:::

## Наступні кроки

Після встановлення модуля перейдіть до [Використання модуля](./usage.md) для ознайомлення з функціоналом.
