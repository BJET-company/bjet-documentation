# Платежі LiqPay на Odoo вебсайті

**Технічна назва модуля:** `bjet_liqpay_payment`

## Для чого цей модуль?

**Модуль Платежі LiqPay на Odoo вебсайті** додає в Odoo платіжного провайдера **LiqPay** для онлайн-оплат:
- при оформленні замовлень на сайті (eCommerce checkout);
- за посиланнями на оплату (Payment Link) з рахунків (Invoice) та замовлень на продаж (Sales Order).

Після успішної оплати в Odoo автоматично створюються та зв’язуються між собою:
- Рахунок (Invoice);
- Платіжна транзакція (Payment Transaction);
- Платіж (Payment).

У платіжній транзакції та платіжному документі зберігається **унікальний ID платежу LiqPay** для подальшого узгодження банківських виписок.
