# Встановлення та активація

1. **Завантажте модуль**

   Завантажте архів bjet_import_nbu_exchange_rate.zip в Odoo

2. **Активуйте режим розробника**

   Загальні налаштування → Інструменти розробника → Активуйте режим розробника.

3. **Оновіть список модулів**

   Натисніть кнопку Оновити список додатків (Update Apps List).

4. **Встановіть модуль**

   Знайдіть Import NBU Exchange Rate та натисніть Активувати.

## Права доступу користувачів

| Група доступу | Можливості |
|--------------|------------|
| Адміністрування/Налаштування | Доступ до налаштувань імпорту курсів валют у формі компанії |
| Бухгалтерія/Виставлення рахунків | Виконання імпорту курсів валют через майстер імпорту |
| Технічні функції | Перегляд поля планувальника (cron) у режимі розробника |
