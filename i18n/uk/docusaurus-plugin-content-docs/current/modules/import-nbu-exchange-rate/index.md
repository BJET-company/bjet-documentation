# BJet Курси валют НБУ

## Опис модуля

Модуль Курси валют НБУ дозволяє автоматично або вручну завантажувати курси валют НБУ в Odoo 18 (Community/Enterprise).

## Призначення та можливості

Модуль призначений для компаній, які хочуть:

- Автоматично імпортувати курси валют з Національного банку України;
- Вручну завантажувати курси валют за потрібний період;
- Налаштувати автоматичне оновлення курсів валют за розкладом.

## Версія модуля

**Версія:** 18.0.1.0.2
**Підтримувані версії Odoo:** 18.0 Community/Enterprise
