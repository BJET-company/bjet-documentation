# Усунення несправностей

## Проблема: Імпорт курсів не виконується

### Можливі причини:

1. **Не вибрано сервіс курсів валют**
   - Перевірте, чи заповнене поле "Сервіс курсів валют" у налаштуваннях компанії
   - За замовчуванням має бути вибрано "НБУ"

2. **Відсутнє з'єднання з інтернетом**
   - Переконайтесь, що сервер Odoo має доступ до інтернету
   - Перевірте налаштування firewall та proxy

3. **API НБУ недоступний**
   - Перевірте доступність сервісу НБУ
   - Спробуйте виконати імпорт пізніше

## Проблема: Курси валют не оновлюються автоматично

### Рішення:

1. **Перевірте cron job**
   - Увійдіть у режим розробника
   - Перейдіть до налаштувань компанії
   - Перевірте статус планувальника у полі "Автоматичне завантаження"
   - Активуйте cron, якщо він неактивний

2. **Перевірте налаштування планувальника**
   - Перейдіть до Налаштування → Технічні → Автоматизація → Заплановані дії
   - Знайдіть дію для імпорту курсів валют
   - Перевірте інтервал виконання та статус

## Проблема: Помилка при імпорті

### Рішення:

1. **Перевірте повідомлення про помилку**
   - Система відображає повідомлення про помилку безпосередньо в інтерфейсі
   - Прочитайте текст помилки для розуміння причини

2. **Перевірте права доступу**
   - Переконайтесь, що у користувача є права групи "Бухгалтерія/Виставлення рахунків"

3. **Перевірте налаштування валют**
   - Перейдіть до Бухоблік → Налаштування → Валюти
   - Переконайтесь, що необхідні валюти активовані в системі

4. **Перевірте логи сервера** (тільки для адміністраторів)
   - Модуль записує помилки у серверні логи Odoo
   - Перегляньте файл логів для детальної інформації про помилки
