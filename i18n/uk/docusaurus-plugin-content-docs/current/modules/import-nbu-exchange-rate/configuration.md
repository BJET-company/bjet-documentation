# Налаштування модуля

## Доступ до налаштувань

Після встановлення модуля перейдіть до: **Налаштування → Компанія → Відкрити форму компанії**

У формі компанії з'явиться нова вкладка **Імпорт курсів валют**.

![Форма компанії з вкладкою Імпорт курсів валют](/img/import-nbu-exchange-rate/company-form-import-tab-uk.png)

## Поля налаштувань

### Сервіс курсів валют

Виберіть сервіс, з якого будуть завантажуватися курси валют. За замовчуванням встановлено **НБУ**.

:::warning Важливо
Поле можна залишити порожнім, проте в цьому випадку імпорт курсів не буде виконаний.
:::

### Період імпорту курсів валют

Вкажіть інтервал у днях, за який потрібно імпортувати курси.

При ручному імпорті система автоматично розраховує дату початку та дату закінчення імпорту на основі цього числа та поточної дати. Значення можна змінити вручну.

### Автоматичне завантаження

Планувальник (cron), який виконує автоматичне завантаження курсів валют.

:::info Примітка
- Cron створюється автоматично для кожної компанії після збереження налаштувань імпорту валютних курсів
- Поле видно лише у DEV режимі
- За замовчуванням cron неактивний
:::

## Ручний імпорт курсів валют

### Крок 1: Запуск імпорту

Для ручного імпорту натисніть кнопку **Імпорт курсів валют**

![Кнопка Імпорт курсів валют](/img/import-nbu-exchange-rate/import-button-uk.png)

### Крок 2: Налаштування періоду

Якщо потрібно, ви можете змінити період завантаження валютних курсів.

Натисніть **Імпортувати**

![Майстер імпорту з налаштуванням діапазону дат](/img/import-nbu-exchange-rate/import-wizard-uk.png)

### Крок 3: Перевірка результатів

Перейдіть у меню **Бухоблік → Налаштування → Валюти**.

Курси валют завантажено у систему.

![Список валют з імпортованими курсами](/img/import-nbu-exchange-rate/currency-rates-list-uk.png)
