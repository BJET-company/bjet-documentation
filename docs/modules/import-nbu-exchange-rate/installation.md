# Installation and Activation

1. **Download the module**

   Download the bjet_import_nbu_exchange_rate.zip archive to Odoo

2. **Activate developer mode**

   General Settings → Developer Tools → Activate developer mode.

3. **Update module list**

   Click the Update Apps List button.

4. **Install the module**

   Find Import NBU Exchange Rate and click Activate.

## User Access Rights

| Access Group | Capabilities |
|--------------|------------|
| Administration/Settings | Access to currency import settings in company form |
| Accounting/Invoicing | Execute currency import through import wizard |
| Technical Features | View scheduler (cron) field in developer mode |
