# Installation and Activation

## Module Installation

1. Go to the **Apps** menu.

2. Find the **LiqPay Subscription Payments** module.

3. Click **Install**.

:::info Note
The **LiqPay Odoo Subscription Payments** module extends the functionality of the **LiqPay Payments on Odoo Website** module, which must be installed and configured to enable automated subscription payments via LiqPay.
:::

## Next Steps

After installing the module, proceed to [Configuration](./configuration.md) to learn about the functionality.
