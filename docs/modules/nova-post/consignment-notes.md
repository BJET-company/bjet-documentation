# Section 5: Consignment Note Management

The **Consignment Notes** model allows centralized management of all shipments.

![Nova Poshta Consignment Notes list](/img/nova-post/consignment-notes-en.png)

1. Navigate to **Nova Post** ➔ **Consignment Notes** (available in Sales, Purchase, and Inventory).
2. Detailed information available:
   - **Direction:** Incoming (Purchases) or Outgoing (Sales).
   - **COD Data:** Amount, payer, and payout status.
   - **Status History:** Last updated time of the shipment.
3. **Refresh Status:** Manually fetch the latest API data regardless of the sync schedule.
4. **Track on Nova Poshta:** Opens the official Nova Poshta tracking page for that specific TTN.
