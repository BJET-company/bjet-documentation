# Section 2: Order and Warehouse Workflow

## 2.1. Waybill Visibility in Sales and Purchase Lists (Tree View)

The **Sales** and **Purchase** modules include columns for quick monitoring of waybills:

- **Tracking Reference:** Displays all TTN numbers associated with the order.
- **Carrier:** The selected delivery method.
- **Status:** Color-coded badges showing the current delivery state (e.g., "Arrived at branch" or "Received").

You can search for orders by TTN number or group them by status using the standard search bar.
