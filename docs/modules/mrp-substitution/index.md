# MRP Component Substitution with Priority Levels

## What Is This Module For?

In manufacturing, a production stop caused by material shortages leads to:
- missed deadlines
- equipment downtime
- additional costs
- dissatisfied customers

This module allows you to automatically replace out-of-stock components with predefined substitutes, taking into account priority levels and substitution ratios.

## What You Gain

- Production does not stop due to shortages
- The system automatically selects substitutes based on priority
- Partial replacement using multiple substitutes is supported
- Full transparency — all changes are recorded in the Manufacturing Order
- Control over substitute priority levels
