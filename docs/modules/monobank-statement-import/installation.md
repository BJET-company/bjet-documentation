# Installation and Activation

## Module Installation

1. Go to the **Apps** menu.

2. Search for the **Monobank Statement Import** module.

3. Click **Install**.

:::info Note
If the Accounting [account] module is not yet installed, it will be installed automatically.
:::

## Access Rights

Users belonging to the **Accounting > Administrator** group can configure the Monobank integration:

- Fill in Monobank fields in the Journal;
- Run manual import;
- Manage scheduled actions (cron) if technical access is available.

## Next Steps

After installing the module, proceed to [Configuration](./configuration.md) to set up the Monobank integration.
