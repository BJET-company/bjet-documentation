# Installation and Activation

## Module Installation

1. Go to the **Apps** menu.

2. Find the **LiqPay Website Payments** module.

3. Click **Install**.

:::info Note
If the Invoicing [account] or Payment Engine [payment] modules are not yet installed, they will be installed automatically.
:::

## Next Steps

After installing the module, proceed to [Configuration](./configuration.md) to learn about the functionality.
