# Installation and Activation

## Module Installation

1. Go to the **Apps** menu.

2. Find the **Manufacturing BOM Structure (MO Hierarchy Tree)** module.

3. Click **Install**.

:::info Note
If the Manufacturing [mrp] module is not yet installed — it will be installed automatically.
:::

## Verifying the Installation

After installing the module, new tabs will appear:

1. Go to **Manufacturing → Bills of Materials** and open any Bill of Materials — a new **BOM Structure** tab should appear.
2. Go to **Manufacturing → Manufacturing Orders** and open any Manufacturing Order — a new **MO Hierarchy Tree** tab should appear.
