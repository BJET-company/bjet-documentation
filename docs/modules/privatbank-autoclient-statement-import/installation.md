# Installation and Activation

## Module Installation

1. Go to the **Apps** menu.

2. Find the **PrivatBank Autoclient Statement Import** module.

3. Click **Install**.

:::info Note
If the Accounting [account] module is not yet installed â€” it will be installed automatically.
:::

## Access Rights

Users who belong to the **Accounting > Administrator** group can configure the PrivatBank integration:

- Fill in PrivatBank fields in the Journal;
- Run import manually;
- Manage schedulers (if technical access is available).

## Next Steps

After installing the module, proceed to [Configuration](./configuration.md) to configure the PrivatBank integration.
